## Step 1: Upload the Sample Excel File
**Description:** Before writing any prompt, upload the sample Excel file to Claude. The file should contain columns: Name, Designation, Manager Name, and Profile Picture URL.

**Callout:** Upload the Excel file before giving Claude the build prompt.

## Step 2: Write the Build Prompt
**Description:** Once the Excel file is uploaded, paste the build prompt into Claude. The prompt asks Claude to create an Org Chart Generator app that reads the Excel data and renders a visual hierarchy.
**AI Message:** Now paste the build prompt into Claude. Make sure the Excel file is already attached before you send. The prompt tells Claude exactly what to build.
**Callout:** Paste the build prompt after the file is uploaded.
**Slide:** 2

## Step 3: Review the Generated Output
**Description:** Claude will generate the org chart application. Review the result carefully — check that names, designations, and hierarchy are correct. Ask Claude to fix anything that looks wrong.
**AI Message:** Claude has generated the app! Review it carefully. If anything looks off — wrong hierarchy, missing names, layout issues — just tell Claude what to fix in plain English.
**Callout:** Review the generated result and ask Claude to fix anything wrong.
**Slide:** 3

## Step 4: Publish the App
**Description:** Once the org chart looks correct, publish it using Claude's Share or Publish feature. Copy the link to share with colleagues.
**AI Message:** Looking good! Now let's publish the app. Look for the Publish or Share button in Claude's interface and copy the link.
**Callout:** Look for Publish, Share, or Copy Link.
**Slide:** 4

## Step 5: Test the Published App
**Description:** Open the published link, upload the Excel file, and verify the org chart generates correctly. Share the link with your team.
**AI Message:** Final step — open the published link and test it yourself. Upload the Excel file and confirm the org chart looks correct. Then share the link with your colleagues!
